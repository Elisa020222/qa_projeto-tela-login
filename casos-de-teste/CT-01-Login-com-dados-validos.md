# CT-01 - Login com dados válidos

## 📌 Objetivo
Validar que o sistema permite acesso quando o usuário informa email e senha corretos.

## 🔎 Pré-condição
Usuário previamente cadastrado no sistema.

## 🧪 Passos para execução
1. Acessar a tela de login.
2. Inserir um email válido.
3. Inserir uma senha válida.
4. Clicar no botão "Entrar".

## ✅ Resultado Esperado
O usuário deve ser redirecionado para a página inicial do sistema.
