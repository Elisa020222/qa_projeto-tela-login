# CT-03 - Login com campos vazios

## 📌 Objetivo
Validar que o sistema impede o login quando os campos obrigatórios não são preenchidos.

## 🔎 Pré-condição
Usuário na tela de login.

## 🧪 Passos para execução
1. Acessar a tela de login.
2. Não preencher o campo email.
3. Não preencher o campo senha.
4. Clicar no botão "Entrar".

## ❌ Resultado Esperado
O sistema deve exibir mensagens informando que os campos são obrigatórios.
O usuário não deve conseguir acessar o sistema.
