# CT-02 - Login com senha inválida

## 📌 Objetivo
Validar que o sistema não permite acesso quando a senha informada está incorreta.

## 🔎 Pré-condição
Usuário previamente cadastrado no sistema.

## 🧪 Passos para execução
1. Acessar a tela de login.
2. Inserir um email válido.
3. Inserir uma senha inválida.
4. Clicar no botão "Entrar".

## ❌ Resultado Esperado
O sistema deve exibir a mensagem:
"Email ou senha inválidos."
O usuário não deve acessar o sistema.
