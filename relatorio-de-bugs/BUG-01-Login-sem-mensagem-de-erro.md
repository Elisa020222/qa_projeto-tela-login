# BUG-01 - Sistema não exibe mensagem ao inserir senha inválida

## 📌 Descrição
Ao tentar realizar login com senha incorreta, o sistema não exibe mensagem de erro para o usuário.

## 🌎 Ambiente
- Navegador: Google Chrome
- Sistema Operacional: Windows
- Ambiente: Teste

## 🧪 Passos para Reproduzir
1. Acessar a tela de login.
2. Inserir email válido.
3. Inserir senha inválida.
4. Clicar no botão "Entrar".

## ❌ Resultado Atual
A página apenas recarrega sem exibir mensagem de erro.

## ✅ Resultado Esperado
O sistema deve exibir a mensagem:
"Email ou senha inválidos."

## 🔥 Severidade
Média

## 📊 Status
Aberto
