# Projeto de Teste Manual - Funcionalidade de Login

## 📌 Objetivo
Este projeto simula a execução de testes manuais na funcionalidade de login de uma aplicação web.

O objetivo foi aplicar na prática conceitos de:
- Criação de casos de teste
- Cenários positivos e negativos
- Validação de campos obrigatórios
- Teste exploratório
- Registro e documentação de bugs

---

## 🧪 Casos de Teste Criados

- CT-01: Login com dados válidos
- CT-02: Login com senha inválida
- CT-03: Login com campos vazios
- CT-04: Teste exploratório

---

## 🐞 Bugs Registrados

- BUG-01: Sistema não exibe mensagem ao inserir senha inválida

---

## 🛠 Técnicas Aplicadas

- Teste manual
- Documentação estruturada
- Análise de comportamento do sistema
- Classificação de severidade
